# Clans Fabric

This is a Clans port for Fabric server platform.  
Very incomplete, WIP.

## Usage
```bash
git clone https://github.com/whitebelyash/Clans-Fabric && cd Clans-Fabric && ./gradlew build
```
Output file is `build/libs/clans-fabric-$VERSION-full-remapped.jar`.
